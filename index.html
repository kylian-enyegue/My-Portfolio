Project Title

This project consists of a theoretical presentation and a practical simulation on Cisco Packet Tracer of DDoS attacks. By combining performance tracking with personalized metabolic calculations, it provides users with a clear overview of their progress and training intensity.

Player Stats Companion App

This mobile application is a Kotlin-based companion app that uses the SoFIFA player database via an API to help EA FC 26 Career Mode players. It lets users quickly search football players and view key information such as overall rating, potential, positions, physical attributes, market value, and wage. The goal is to make scouting and transfer decisions easier by centralizing reliable player data in a clean, fast interface. By consuming the API and displaying updated statistics in real time, the app provides a practical tool for building squads, comparing profiles, and optimizing career mode strategies.
